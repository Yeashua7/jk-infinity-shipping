# Sitio Web Encomiendas USA-SV

## Archivos Incluidos:
- `index.html` - Estructura principal del sitio web
- `styles.css` - Estilos y diseño responsive
- `script.js` - Funcionalidades JavaScript

## Instrucciones de Instalación:

1. **Descomprimir archivos**: Extrae todos los archivos en una carpeta
2. **Abrir en navegador**: Haz doble clic en `index.html`
3. **Personalizar**: Edita los archivos según tus necesidades

## Personalización Requerida:

### Redes Sociales (en index.html):
- Cambiar `tuempresa` por tu nombre real en Facebook, Instagram y TikTok
- Cambiar `1234567890` por tu número de WhatsApp real

### Información de Contacto:
- Actualizar direcciones en el footer
- Cambiar números de teléfono
- Modificar email de contacto

### Colores y Branding:
- Editar variables CSS en `styles.css` (líneas 20-30)
- Cambiar nombre de empresa en el logo
